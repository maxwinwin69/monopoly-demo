## 玩法

1. **設定下注** — 調整下注星幣（$10 ～ $500）
2. **🚀 發射** — 擲出兩顆骰子，棋子沿 16 格棋盤移動
3. **收集獎金** — 落在房子格，獎金自動累積進**宇宙能量池**
4. **承擔風險** — 落在黑洞/蟲洞格，直接扣除餘額
5. **決策時刻** — 經過出發點時選擇：
   - **🛸 傳送入帳** — 立即領走獎池
   - **繼續** — 再繞一圈，提領倍率 +0.5×

### 繞圈倍率機制

越多圈不提領，倍率越高，但風險也越大：

| 圈數 | 提領倍率 | 視覺張力 |
|:----:|:-------:|:-------:|
| 1 | ×1.0 | 🟢 綠色 |
| 2 | ×1.5 | 🟡 黃綠 |
| 3 | ×2.0 | 🟡 黃色 |
| 4 | ×2.5 | 🟠 橘色 ⚠ |
| 5+ | ×3.0+ | 🔴 紅色 危險 |

## 特效

- **粒子系統** — 每種事件有專屬爆炸效果（獲獎、大獎、危險、提領）
- **棋子動畫** — 外星綠色發光球，旋轉虛線軌道環
- **圈數視覺** — 中央獎池字體逐圈放大、顏色漸變、旋轉能量環
- **環境火花** — 圈數越高，中央面板自動噴出粒子
- **螢幕震動** — 落在危險格時觸發
- **動態星空** — 中央面板有閃爍星點與掃描線
- **背景音樂** — Web Audio API 合成的太空琶音，自動循環
- **音效** — 骰子、移動、獲獎、JACKPOT、危險、提領各有專屬聲效

## 技術

- 純 HTML / CSS / JavaScript，零依賴
- Canvas 2D 繪圖（棋盤、棋子、粒子）
- Web Audio API（背景音樂 + 音效，無外部音檔）
- 手機直版優化（`100dvh`，禁止縮放）
- `requestAnimationFrame` 動畫主迴圈（~36fps）
